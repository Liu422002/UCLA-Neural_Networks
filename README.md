# UCLA Admission Project

## Overview
This project predicts whether a student will be admitted to UCLA using a machine learning model.

## Models Used
- MLP Classifier (Neural Network)

## Project Structure
- `src/`: modularized Python files
- `data/`: dataset
- `models/`: saved trained model and scaler
- `app.py`: Streamlit app

## How to Run
1. Install dependencies:
   pip install -r requirements.txt

2. Train the model:
   python main.py

3. Run the Streamlit app:
   streamlit run app.py
